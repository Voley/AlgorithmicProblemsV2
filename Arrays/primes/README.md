# Count Primes (Easy 204)
Count the number of prime numbers less than a non-negative number, n.

# Solution:

<details><summary>Click here</summary>  
Allocate boolean array from 0 to n, then iterate over it from 2 to sqrt(n) + 1
and remove everything that is multiplier of that value.
O(n) time, O(n) space

<br></br>

</details>
